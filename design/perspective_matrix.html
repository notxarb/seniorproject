Perspective Matrix

Definitions:

Perspective Matrix - Matrix used to adjust 3 dimensional points to a 2 dimensional space, allowing for further points to be adjusted differently than closer ones.

Descriptions:

This uses a truncated frustrum pyramid to calculate the projection of 3 dimensions on to 2 dimensions. This will need the following attributes:

  • perspective matrix - This will be a 4x4 array that holds the frustrum matrix.
This will need to have the following functions:
  • Frustrum - This function is used to create the frustrum matrix.
  • Create Perspective Matrix - This will take input that comes from the form, put it in the create frustrum, and save it as its matrix.
  • To GL Matrix - WebGL wants the values in a 1 dimensional array, reading down the columns.
