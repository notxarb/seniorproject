Directional Lighing

Definitions:

Directional Lighting - Parallel light beams that share a common vector for direction.

Descriptions:

This will hold data to be pulled into the vertex shader. It will need to have the following property.

  • Color - An array holding floating point values representing red, green, and blue. Vector - An array holding floating point values representing x, y, and z components of a vector.
This will need the following function:
  • Unit Vector - This will take the value in vector and calculate a unit vector for it. This is used when passing in the light direction to the vertex shader.
