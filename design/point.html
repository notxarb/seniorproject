Point

Definitions:

Point - A representation of a position in 3 space using coordinates x, y, and z.

Descriptions:

A surface is comprised of 3 or more points. Point generation for an implicit function will generate values by iterating across the range of x values. For each x, it will iterate across the range of y values. For each x, y pair, it will generate values for z using the supplied function. Point generation for parametric equations will generate values by iterating across the range of u values. For each u, it will iterate across the range of v values. For each u, v pair, it will generate values for x, y, and z using the supplied functions.
