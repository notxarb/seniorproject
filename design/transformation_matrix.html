Transformation Matrix

Definitions:

Transformation Matrix - A matrix that is used to do a combination of translation and rotation operations in 3 space.

Descriptions:

This will take input from the settings panel and generate a matrix for WebGL to use for rotation of the graph. general rotations yaw, pitch and roll. May be fun to see this matrix.

This will need to have the following functions:

  • Multiply Matrix - This function is used to combine the rotations and transposing operations.
  • Rotate - This needs to be able to take x, y, and z rotations and multiply them together.
  • Translate - This needs to be able to take x, y, and z and make linear translations.
  • Create Transformation Matrix - This needs to take the rotations and translations and multiply them together to generate the final one.
  • To GL Matrix - WebGL wants the values in a 1 dimensional array, reading down the columns.
  • Create Normal Matrix - This will make a matrix perpendicular to the transformation matrix. It's important for transforming the directional lighting.
