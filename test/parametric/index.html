Test 1 - Cone

  • x = v * Math.cos(u)
  • y = v * Math.sin(u)
  • z = v

Test 2 - Torus

  • x = Math.cos(u) * ( 6 + 3 * Math.cos(v))
  • y = Math.sin(u) * (6 + 3 * Math.cos(v))
  • z = 3 * Math.sin(v)

Test 3 - Elipsoid

  • x = 6 * Math.cos(u) * Math.cos(v)
  • y = 4 * Math.sin(u) * Math.cos(v)
  • z = 3 * Math.sin(v)

Test 4 - Möbius Strip

  • x = 2 * ( Math.cos( u ) + v / ( 2 * Math.PI) * Math.cos( u / 2 ) * Math.cos(u))
  • y = 2 * ( Math.sin( u ) + v / ( 2 * Math.PI ) * Math.cos( u / 2 ) * Math.sin(u))
  • z = v / Math.PI * Math.sin( u / 2 )

Test 5 - Helicoid

  • x = v * Math.cos(u)
  • y = v * Math.sin(u)
  • z = u
