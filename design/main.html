Main

Definitions:

Main - Commonly used in programming to hold main flow of a program.

Descriptions:

This will take care of basic set up of the WebGL. This will also take care of user input and adjust other objects to account for desired changes. This will need to have the following attributes:

  • GL - This is the reference to interact with the WebGL binding from the canvas.
  • Transformation Matrix - This is used to store rotation and translation information.
  • Perspective Matrix - This is used to calculate how to adjust the image to give perspective to the objects.
  • Ambient Light - This stores the information on ambient lighting to use.
  • Directional Light - This stores the information needed to calculate the directional lighting.
  • Shader Program - This is where the vertex and fragment shaders are registered with WebGL.
  • Equation - This is where the equation data is stored, points are generated, and other things related to the graph.
Main will need the following functions:
  • On Page Load - This takes an equation as a string, or an array and passes it to equation. It also initializes all other objects needed to display a graph.
  • Initialize Canvas - This will get the canvas ready to run WebGL.
  • Initialize Transformation Matrix - This will get initial values from the form and initialize the transformation matrix object.
  • Initialize Perspective Matrix - This will get initial values from the form and initialize the perspective matrix object.
  • Initialize Ambient Light - This will get initial values from the form and initialize the ambient lighting object.
  • Initialize Directional Light - This will get initial values from the form and initialize the directional lighting object.
  • Get Shader - This will make a web request to get the glsl files for the vertex and fragment shaders.
  • Initialize Shaders - This calls the Get Shader function and registers the shaders with the WebGL binding.
  • Update Transformation Matrix - This will get updated values from the form and update the transformation matrix object.
  • Update Perspective Matrix - This will get updated values from the form and update the perspective matrix object.
  • Update Ambient Light - This will get updated values from the form and update the ambient lighting object.
  • Update Directional Light - This will get updated values from the form and update the directional lighting object.
  • Set Matrix Uniforms - This binds values from the transformation matrix, perspective matrix, ambient lighting, and directional lighting with the vertex shader.
