Equation

Definitions:

Equation - representation of equality.

Descriptions:

This is where the function(s) that define the graph would be. The equation would potentially have several surfaces. Equations will need to have the following functions:

  • Asymptote detection - This will determine how many surfaces the equation will need to have.
  • Point generation - This would be a good place for the points to be created, and then sorted out into surfaces as the asymptote detection decides.
  • Surface generation - This create the surfaces and determine which points belong.
This will also need to know if the equation is implicit or parametric, and have either a single equation for z, or have an equation for each x, y, and z. It will need references to its generated points, and surfaces.
