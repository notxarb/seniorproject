Canvas

Definitions:

Canvas is the HTML 5 element that displays graphical output to the user.

Descriptions:

In order to use the canvas with WebGL, it will need to have the following functions:
  • Canvas Initializer - This will set up the canvas to have WebGL bindings.
  • Shader Initializer - This will get shaders ready to be used within the rest of the program.
  • Draw - This will call the draw function from the equations, and make sure the transformation matrix and shaders are used.
