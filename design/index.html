Requirements

ID Functional Requirement Demonstration Scenario Success
1 Shall create 3d surface from implicit functions. Shall include several (at least 5) implicit functions to test. Comparison can be made with Google, Wolfram Alpha, or possibly others. 80%
2 Shall create 3d surface from parametric functions. Shall include several (at least 5) parametric functions to test. 80%
3 Shall identify asymptotes and split surfaces as appropriate. Shall include several (at least 5) functions with asymptotes to test. 80%
4 Shall have ambient lighting. Everything at least a little bit visible. 80%
5 Shall have directional lighting. Lighting from a point, illuminating graph, should see things like shadows, variances in illumination according to angle of intersection. 80%
6 Will allow user to input equations. Graph will display according to user inputted data, not just test cases. 80%
7 Shall allow user to rotate graph. User input will allow graph to be rotated, visually observed. 80%
8 Will allow user to set graphing bounds. Will truncate graph when values exceed defined bounds (x, y, z) 80%
9 Will create a transform matrix according to rotation. Will visibly rotate graph. 80%
10 Will allow user to alter zoom of graph. View will get closer or farther away. 80%
11 Will allow user to set color for graph. Graph will be colored according to selected color. 80%
12 Will allow multiple equations on the same canvas. Multiple independent surfaces will be displayed, possibly intersecting. 80%
13 Will show calculated data points on graph. Values will be displayed on the graph. 80%
