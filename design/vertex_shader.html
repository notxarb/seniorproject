Vertex Shader

Definitions:

Descriptions:

This will be writen in GLSL. This sets up the vertexes with properties required for rendering. It will caclulate color and lighting values that are passed to the fragment shader. This will need the following properties:

  • Vertex Position - This is passed in as a bound buffer object. WebGL will automatically iterate through the buffer array.
  • Vertex Normal - This is passed in as a bound buffer object. WebGL will automatically iterate through the buffer array.
  • Translation Matrix - This is passed in with other uniform data. This is generated from a transformation matrix.
  • Perspective Matrix - This is passed in with other uniform data. This is generated from a perspective matrix.
  • Translation Normal Matrix - This is passed in with other uniform data. This is generated from a transformation matrix.
  • Ambient Light Color - This is passed in with other uniform data. This is set in the form data.
  • Directional Light Color - This is passed in with other uniform data. This is set in the form data.
  • Directional Light Direction - This is passed in with other uniform data. This is set in the form data.
